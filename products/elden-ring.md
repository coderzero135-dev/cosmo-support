# Elden Ring Cheats

## PC Mods & Trainers

> **Warning:** Offline mode only. Using mods online will result in a ban.

### Recommended Tools
- **Cheat Engine** - Memory editing for runes, HP, stamina
- **Elden Ring Ultimate Trainer** - All-in-one trainer
- **Flawless Widescreen** - Ultrawide support + FPS unlock
- **Seamless Co-op Mod** - Full co-op without restrictions

### Useful Console Commands (Debug Mode)
Requires modding to enable debug console.

```
/setrunes 99999999       - Max runes
/god                     - God mode
/killall                 - Kill all nearby enemies
/noclip                  - Fly through walls
/teleport [x] [y] [z]    - Teleport to coordinates
```

## Glitches & Exploits

### Rune Farm Method
1. Go to Mohgwyn Palace Approach Ledge-Road
2. Use Gold Scarab talisman + Gold-Pickled Fowl Foot
3. Kill Albinaurics with AoE → ~50k runes per run

### Boss Skip (Mogh)
1. Stand outside fog wall
2. Use specific jump attack at corner
3. Clip through and skip the fight
