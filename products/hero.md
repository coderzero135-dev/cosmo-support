## Errors & Fixes

| Error | Fix |
|-------|-----|
| No menu (Valorant) | Install Medal.TV, ensure record-with-medal is on. Valorant Launcher must NOT be open during inject. |
| No menu (general) | <a href="https://gofile.io/d/sOW8lX" target="_blank">Download this</a>, run as admin. Creates <code>bundle.enc</code> — send in ticket. |
| Secure Boot policy fix | BIOS → Secure Boot → Key Management → Forbidden Signatures (dbx) → Delete. Press No. Delete signatures 1 by 1. Secure Boot mode: Custom. Do NOT press DELETE ALL. |
| BSOD for Apex | <a href="https://go.microsoft.com/fwlink/?linkid=2335869" target="_blank">Download this</a>. File path: <code>C:\Program Files (x86)\Windows Kits\10\</code>. Always generate a new build. |
| External Overlay error | 1. Discord overlay off<br>2. OBS: close or switch to Window Capture<br>3. GeForce Experience overlay off<br>4. Xbox Game Bar off |
| Aimbot or ESP not working | Aimbot: select default preset. ESP: reload cheat within menu. |
| Loader crashes after key verification | Install latest VC++ Redistributable (x86 & x64):<br><a href="https://aka.ms/vs/17/release/vc_redist.x64.exe" target="_blank">x64</a> · <a href="https://aka.ms/vs/17/release/vc_redist.x86.exe" target="_blank">x86</a> |
| Graphic game crash | Send screenshots:<br><a href="https://imgur.com/a/Jf9L64C" target="_blank">Graphics</a><br><a href="https://imgur.com/a/ixjO37i" target="_blank">Overlay crash</a> |
| Arena Breakout ENC:ON | Too many reports — change account. Goes away after 12"“24 hours. |
