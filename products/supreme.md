## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Invalid Processed Data 0x0 | Use a VPN — caused by ISP blocking the connection. |
| Error #3B000000 | Turn off Core Isolation and run in CMD as admin:<br><code>bcdedit /set hypervisorlaunchtype off</code> |
| Error Code 0x1 / 0x2 | Uninstall any anti-cheat and restart PC. |
| Error Code 0x3 / 0x7 | Spoofed before inject — restart PC then inject first. |
