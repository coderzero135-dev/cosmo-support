## Building Tips & Tricks

### Quick Build Combos
```
Wall + Ramp  : Mouse4 → Mouse5
Box Fight    : Wall → Floor → Cone → Wall
90s          : Wall → Ramp → Wall → Ramp (repeat)
Turtling     : Box + Cone inside → Heal
```
