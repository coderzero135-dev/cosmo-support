## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off
- Delete VGK, FaceIT, ACE anti-cheat (not ACE if Delta/ABI)

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Driver initialization Error 1 | Disable virtualization if not working. Check Core Isolation. Or run <a href="https://mega.nz/file/uURS0ZgL#gn9i_rBW__80V9uzexA_Cr2vPUPNGQK2aif4qtevXHs" target="_blank">this file</a>. |
| Spoofer | Run cheat first, then spoofer, then launch game. If cheat menu won't appear after spoofer, simply relaunch — cheat is already injected. |
| PUBG | Must be run in DX11 (change DX version in game). |
| Dark and Darker | Add <code>-dx11</code> in Steam launch options. |
| ABI FULL setup | Navigate to <code>...\ABInfinite\ABInfinite\Binaries\Win64</code>. Create <code>guid.txt</code> and paste someone else's ABI account ID into it. Required for the spoofer to work. |
