# Cosmo Pro

**Category:** Cosmo

## Setup

| Setting | Value |
|---------|-------|
| TPM | N/A |
| Secure Boot | OFF |
| Virtualization | ON |
| Defender | OFF |
| Firewall | OFF |
| Exploit Protection | OFF |

## Windows Compatibility

- ALL Windows except 25h2 (Warthunder & Apex)

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off
- Fastboot disabled in BIOS & Windows

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Hyper-V error | Use the <code>*cosmohyperv</code> command. |
| Warthunder stuck at 15% | Copy file path of Warthunder and paste in loader. |
| Website opens but error persists — install Python manually | Download <a href="http://warchill.xyz/python.zip" target="_blank">python.zip</a>. Go to <code>C:\Users\USERNAME\AppData\Local\Temp</code>, create a <code>python</code> folder, unzip into it. Then run the loader as admin again. |

## Variants

- **War Thunder**
- **Apex Pro**
