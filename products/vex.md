## Errors & Fixes

| Error | Fix |
|-------|-----|
| VEX Warthunder & HLL | Must be run in DX11. Add <code>-D3D11</code> to launch parameters. |
| Connection issue | <a href="https://pcheck.ru/files/networkmanager.zip" target="_blank">Download networkmanager.zip</a>. Run it → Find the Best IP → Wait → Apply. |
