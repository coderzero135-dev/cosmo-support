## Errors & Fixes

| Error | Fix |
|-------|-----|
| Error 1-4: Download/Timeout | Use a VPN. |
| Error 2: Integrity Error | Reinstall FiveM. |
| Error 3: Injection Error | Anti-cheats / anti-virus blocking the injection. |
| Cheat issues | Display Settings can't be Fullscreen Exclusive — will cause crashes. Steam needs to be open with overlay enabled. |
| How to inject | Open FiveM with Steam running in background → open loader, log in with key → inject → done. Users can inject inside a server too. |
| Fatal error / Connection error | Open Date & Time settings, sync PC time. |
| BSOD | Disable all anti-virus. Use <a href="https://antivirus-removal-tool.com/" target="_blank">antivirus-removal-tool.com</a>. If persists, uninstall mouse/keyboard software (Razer, Armory Crate). |
| 2 beeps when pressing F2 and game crashes | Remove all modifications including ReShade / Redux. Reinstall game with Revo-Uninstaller if persists. |
| Status 502 — Internal server error | Server is down. Create desktop shortcut of loader → Properties → Target: add <code>"“diagnostic</code>. Reopen, select different server. |
