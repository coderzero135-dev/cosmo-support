## Windows Compatibility

- ALL Windows versions EXCEPT:
- Marvel, PUBG, OW2, Naraka — 22h2 W10 only

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| ARC Raiders | Steam only and DX12 must be used. |
| Error code 2145 / 2146 | Tell a Senior to contact dev — uploaded file was corrupted during upload. |
| Crashing loader | Paste this into CMD as admin then restart PC (change "aaaaa14" — all lowercase):<br><code>wmic computersystem where name="%computername%" call rename name="aaaaa14"</code> |
