**Category:** Products

## Setup

| Setting | Value |
|---------|-------|
| TPM | N/A |
| Secure Boot | OFF |
| Virtualization | OFF |
| Defender | OFF |
| Firewall | OFF |
| Exploit Protection | OFF |

## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| BSOD | Run <code>sfc /scannow</code> then <code>DISM.exe /Online /Cleanup-image /Restorehealth</code> |
| Loader closes at 100% | Sync PC time. |
| bedaisy.sys / easyanticheat.sys error | Run loader before launching the game. |
| I2 Error | Check all anti-virus/anti-cheats. Ensure Core Isolation disabled, visuals installed, loader run as admin. |
