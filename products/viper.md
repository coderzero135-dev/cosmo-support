## Windows Compatibility

- ALL Windows versions

## Requirements

- ALL anti-virus OFF
- Windows Defender / Smartscreen / Core Isolation off (*defendercontrol)
- Disable all overlays (Discord, Nvidia, Overwolf, ReShade, RivaTuner)
- Run Steam as admin · NO launch parameters
- Disable Frame Generation & Nvidia Reflex in games
- No Nvidia filters
- STEAM ONLY (except ARC Raiders)
- Close background apps · disable in startup
- Visuals installed (*visual)
- Windows & GPU drivers up to date
