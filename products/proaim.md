## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off
- AMD: NX Mode enabled in BIOS (if hypervisor error after fix)

## Errors & Fixes

| Error | Fix |
|-------|-----|
| No menu / BSOD | Send the latest loader from <a href="https://t.me/fecloaders" target="_blank">t.me/fecloaders</a>. If still BSOD, send <code>*delay</code>.<br>Still BSOD? Go to <code>C:\Windows\Minidump</code>, find the latest file, copy to desktop, send in ticket. |
| Hyper-V Not Launched Error | Open CMD as admin:<br><code>sfc /scannow</code><br><code>dism /online /cleanup-image /restorehealth</code><br><code>bcdedit /set hypervisorlaunchtype auto</code> |
| Warthunder | Must be run in DX12. |
