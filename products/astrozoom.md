## Windows Compatibility

- Win 10 22h2 → Win 11 23h2

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Loading mem_ctx | Uninstall all anti-cheats/anti-virus: FaceIT, Vanguard, ACE, Kaspersky, McAfee, Malwarebytes. (Try loading cheat from USB.) |
| Context Error #13 / Error 0x2 | Unsupported Windows version. |
| Ntherr / Nreq Error | Check if customer is on a VM/Virtual Desktop. |
| Connection errors / timeout / couldn't resolve hostname | Use a VPN like Warp or Tunnelbear. |
