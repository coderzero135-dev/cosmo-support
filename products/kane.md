| Setting | Value |

## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off
- Nvidia/AMD overlay
- Steelseries (Rust, Apex, Dark and Darker, Midnight Walkers, PUBG)

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Please download and install the latest vc_redist libs | Use command: <code>*kaneamdmenufix</code> |
| SVM / VMX Error | Disable Windows PIN (<code>*pin</code>) then follow <code>*iobit</code>.<br><a href="https://www.youtube.com/watch?v=L-55o0UnHFU" target="_blank">YouTube guide</a> |
| No menu | Update GPU & Windows drivers, install all visuals, disable other overlays. Do NOT run Steam or game as admin. |
| Failed to Map Memory | If user has > 8GB RAM, disable all startup apps in Task Manager. |
