# Cobra

**Category:** Products

## Setup

| Setting | Value |
|---------|-------|
| TPM | N/A |
| Secure Boot | OFF |
| Virtualization | OFF |
| Defender | OFF |
| Firewall | OFF |
| Exploit Protection | OFF |

## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Mapping driver failed | Disable all anti-cheats & anti-virus. Use <code>*defendercontrol</code>. |
| Loader connection issues | Use Proton VPN. |
| Crashing / BSOD | Make sure no-spoofer download is selected. EAC spoofer not working — for BE spoofer, disable TPM. |
| Overlay issues | Disable all overlays — Discord, Steam, Medal, Nvidia/AMD, etc. |
