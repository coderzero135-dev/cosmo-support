## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off
- EFT CHAMS: press F1 in main menu to inject
- ARC Raiders — Steam only

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Steelseries setup | On steelseries.com/gg: General → turn on Sonar. Then SONAR → Shortcuts → bind F7 to Master Volume Up. Press this bind before injection. |
| SVM / VMX Error | Disable Windows PIN (<code>*pin</code>) then follow <code>*iobit</code>. |
| No menu | Update GPU/Windows drivers, install all visuals, disable overlays. Don't run Steam/game as admin. |
| Failed to Map Memory | If user has > 8GB RAM, disable startup apps in Task Manager. |
