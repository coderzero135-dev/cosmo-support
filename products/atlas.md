|---------|-------|

| Exploit Protection | DEFAULT |

## Windows Compatibility

- ALL Windows versions

## Requirements

- UEFI BIOS
- Visuals installed (*visual)
- Core Isolation off

## Errors & Fixes

| Error | Fix |
|-------|-----|
| Meltdown / Spectre protection | <a href="https://www.guru3d.com/download/download-inspectre/" target="_blank">Download InSpectre</a>. Disable Meltdown & Spectre protection. Then go to <code>C:/RSCFG</code> and delete <code>temp</code>. Restart PC and try again. |
| Blue screens | Disable built-in spoofer. |
| Failed to load driver | Open CMD as admin → run <code>sfc /scannow</code>. Check Core Isolation is disabled. |
| Fatal Error — Engine Error | Open RivaTuner and set detection level to "none". |
| PUBG Blackscreen | Delete the Movies folder at <code>C:\Program Files (x86)\Steam\steamapps\common\PUBG\TslGame\Content\Movies</code> |
