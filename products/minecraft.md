# Minecraft Cheats

## Console Commands

Enable cheats when creating a world or use `/gamemode creative`.

### Game Modes
```
/gamemode creative
/gamemode survival
/gamemode adventure
/gamemode spectator
```

### Items & Blocks
```
/give @p minecraft:diamond 64
/give @p minecraft:netherite_sword 1
/give @p minecraft:enchanted_golden_apple 64
```

### World Controls
```
/time set day
/time set night
/weather clear
/weather thunder
```

### Teleportation
```
/tp @p 0 70 0
/tp @p ~ ~100 ~
```

### Effects
```
/effect give @p speed 99999 5
/effect give @p strength 99999 5
/effect give @p night_vision 99999
/effect clear @p
```

| Code | Effect |
|------|--------|
| 1 | Speed |
| 3 | Haste |
| 5 | Strength |
| 8 | Jump Boost |
| 10 | Regeneration |
| 11 | Resistance |
| 12 | Fire Resistance |
| 14 | Invisibility |
| 16 | Night Vision |
